# SceneTextRecognitioniOS
![SceneTextRecognitioniOS](/recognition.gif)

A scene text recognition demo app using Vision framework and tesseract

### Prerequisite
Xcode 9.0 or later and iPhone or iPad running iOS 11 or later.
